MQX™ RTOS Software Solutions

Accelerate your design success with complimentary RTOS, TCP/IP and USB stacks provided by Freescale MQX™ RTOS Software Solutions. Available on Freescale processors for more than 15 years, Freescale MQX™ RTOS Software Solutions offer a straightforward API, which has a modular architecture and is scalable to fit most requirements, making it simple to fine tune custom applications.

The combination of our market-proven Freescale MQX™ RTOS Software Solutions and silicon portfolio provides a streamlined and powerful platform by creating a comprehensive source for hardware, software, tools and services needs.

Freescale MQX™ RTOS Software Solutions are offered together with a strong ecosystem of RTOS providers, giving you the most flexibility when selecting software solutions.

Implementing MQX™ RTOS IEEE 1588 Kinetis

details...

This document describes implementation of the IEEE 1588 V2 Precision Time Protocol (PTP) on the Kinetis K6x processor running on the MQX™ Real Time Operating System (RTOS). The solution applies to TWR-K60N512 KIT, TWR-K60D100M KIT, TWR-K60F120M KIT on sub-family K60 and TWR-K64F120M KIT on sub-family K64. The demo software runs on the Freescale MQX™ RTOS and uses the Freescale MQX™ IEEE 1588 Communication Library. The library is based on the IEEE 1588 V2 protocol software by IXXAT Automation GmbH.
