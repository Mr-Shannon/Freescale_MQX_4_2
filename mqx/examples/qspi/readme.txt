It is a test to validate the functionality of QSPI.

Overall Info:
If the test is passed, it will print "All tests are passed!" in the serial debug port.

Detail Info:
In this example, it will cover the following items:
1. Erase the whole flash
2. Read data via AHB
3. Read data via IP command
4. Write throughput 
5. Read throughput 
6. Comparison test while calling reading using IP command
7. Erase sector
8. Comparison test while calling read_data using AHB for different start address and data length
9. Comparison test while calling read_byte using AHB for different start address and data length
